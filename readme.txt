Enable WP Core Rest API v2 for ThirstyAffiliates data

== Description ==

This plugin enables access to ThirstyAffiliates data through the WordPress Rest API v2 (part of WP core since 4.7).

Two new endpoints will be available:

1. An end point to read/update the ThirstyAffiliate links on /wp-json/wp/v2/thirstylink

2. An end point to read/update the ThirstyAffiliate categories on /wp-json/wp/v2/thirstylink-category

== Installation ==

1.  Extract the zip file and copy contents in the wp-content/plugins/ directory of your WordPress installation
2.  Login as admin and go to Plugins
3.  Activate this plugin
